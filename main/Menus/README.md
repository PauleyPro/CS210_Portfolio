# Menus Folder

This folder organizes the menu system by development stage, validation notes, and grading alignment.

## Structure

- `Stages` contains the menu-related source files used by the program.
- `Tests` stores manual testing notes for each menu option.
- `Requirements` explains how each menu option maps to the assignment rubric.

## Why It Exists

The menu is the part of the program the user interacts with most directly. Keeping the interface, testing notes, and requirement mapping together makes it easier to understand how each menu option was designed and verified.
